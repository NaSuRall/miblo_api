
## Miblo — API Generator for Front‑End Developers
    Miblo est une plateforme pensée pour les développeurs front‑end qui ont besoin d’API rapides, simples et prêtes à l’emploi — sans avoir à coder un backend complet.
    L’objectif est de fournir un outil capable de générer automatiquement des API personnalisées, adaptées aux besoins du projet, avec ou sans données pré-remplies.
